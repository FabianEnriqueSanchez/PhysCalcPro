## Modulos disponibles

| Modulo | Formula | Aplicacion |
|---|---|---|
| Calor | Q = m · c · ΔT | Control de temperatura |
| Temperatura | °C ↔ °F ↔ K | Monitoreo termico |
| Presion | P = F / A | Sistemas hidraulicos |
| Caudal Volumetrico | Q = V / t | Control de flujo |
| Caudal (A×v) | Q = A · v | Diseño de tuberias |
| Energia Potencial | Ep = m · g · h | Sistemas de elevacion |
| Eficiencia | η = (Eutil / Etotal) × 100 | Optimizacion de recursos |
